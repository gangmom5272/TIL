# 혼자 공부하는 네트워크 4강 ~ 7강

## 패킷의 구조

패킷은 택배에 비유할 수 있다.

| 패킷 구성 요소 | 택배 비유 | 의미 |
|---|---|---|
| 페이로드 | 물품 | 전송하려는 데이터 |
| 헤더 | 택배 송장 | 부가 정보 또는 제어 정보 |
| 트레일러 | 택배 송장 | 부가 정보 또는 제어 정보 |

## 헤더에 담기는 정보

헤더에는 송수신지를 특정하는 정보가 담긴다.

예시:

- IP 주소
- MAC 주소

주소가 있으면 누구에게 데이터를 전송할지 지정할 수 있다.

## 송수신지 유형별 전송 방식

### 유니캐스트

유니캐스트(unicast)는 하나의 수신지에 메시지를 전송하는 방식이다.

송신지와 수신지가 일대일로 메시지를 주고받는 가장 일반적인 송수신 형태이다.

### 브로드캐스트

브로드캐스트(broadcast)는 네트워크상의 모든 호스트에게 메시지를 전송하는 방식이다.

브로드캐스트 도메인은 브로드캐스트가 전송되는 범위이다.

### 멀티캐스트

멀티캐스트(multicast)는 네트워크 내의 동일 그룹에 속한 호스트에게만 메시지를 전송하는 방식이다.

### 애니캐스트

애니캐스트(anycast)는 네트워크 내의 동일 그룹에 속한 호스트 중 가장 가까운 호스트에게 메시지를 전송하는 방식이다.

## 프로토콜

프로토콜(protocol)은 노드 간 정보를 올바르게 주고받기 위해 합의된 규칙이나 방법이다.

특징:

- 통신 과정에서는 일반적으로 여러 프로토콜이 함께 사용된다.
- 각 프로토콜에는 서로 다른 목적과 특징이 있다.
- 프로토콜마다 패킷의 헤더 구조가 다를 수 있다.

## 네트워크 참조 모델

네트워크를 통해 송수신하는 과정에는 정형화된 여러 단계가 있다.

이 단계는 계층적으로 표현할 수 있다.

네트워크 참조 모델은 통신이 이루어지는 각 과정을 계층으로 나눈 구조이다.

### 통신 과정을 계층으로 나누는 이유

1. 네트워크 구성과 설계가 쉬워진다.
2. 네트워크 문제 진단과 해결이 쉬워진다.

## OSI 모델

OSI 모델은 국제 표준화 기구 ISO에서 만든 네트워크 참조 모델이다.

OSI 모델은 7계층으로 구성된다.

| 계층 | 역할 |
|---|---|
| 응용 계층 | 사용자 및 응용 프로그램에 다양한 네트워크 서비스를 제공 |
| 표현 계층 | 문자 인코딩, 압축, 암호화 등을 담당 |
| 세션 계층 | 응용 프로그램 간 연결 상태인 세션을 관리 |
| 전송 계층 | 신뢰성 있고 안정적인 전송을 담당 |
| 네트워크 계층 | 다른 네트워크에 속한 수신지까지 메시지를 전달 |
| 데이터 링크 계층 | 같은 네트워크 내 주변 장치 간 정보를 올바르게 전달 |
| 물리 계층 | 1과 0으로 표현되는 비트 신호를 주고받음 |

## TCP/IP 모델

TCP/IP 모델은 인터넷 프로토콜 스위트 또는 TCP/IP 프로토콜 스택이라고도 한다.

| TCP/IP 계층 | OSI 모델과의 관계 |
|---|---|
| 응용 계층 | OSI의 세션, 표현, 응용 계층과 유사 |
| 전송 계층 | OSI의 전송 계층과 유사 |
| 인터넷 계층 | OSI의 네트워크 계층과 유사 |
| 네트워크 액세스 계층 | OSI의 데이터 링크 계층과 유사 |

네트워크 액세스 계층은 링크 계층 또는 네트워크 인터페이스 계층이라고도 한다.

## 캡슐화와 역캡슐화

송신 과정에서는 메시지가 가장 높은 계층에서 가장 낮은 계층으로 이동한다.

수신 과정에서는 메시지가 가장 낮은 계층에서 가장 높은 계층으로 이동한다.

### 캡슐화

캡슐화는 계층별 프로토콜의 목적과 특징에 맞는 헤더 또는 트레일러가 추가되는 과정이다.

상위 계층의 헤더와 페이로드는 다음 하위 계층의 페이로드가 된다.

트레일러는 주로 데이터 링크 계층에서 붙는다.

### 역캡슐화

역캡슐화는 계층별 프로토콜의 목적과 특징에 맞게 헤더 및 트레일러를 확인한 뒤 제거하는 과정이다.

## PDU

PDU(Protocol Data Unit)는 각 계층에서 송수신되는 메시지의 단위이다.

현재 계층의 PDU는 상위 계층의 데이터에 현재 계층의 프로토콜 헤더 및 트레일러가 붙은 것이다.

| OSI 계층 | PDU |
|---|---|
| 응용 계층 | 데이터 |
| 표현 계층 | 데이터 |
| 세션 계층 | 데이터 |
| 전송 계층 | TCP: 세그먼트, UDP: 데이터그램 |
| 네트워크 계층 | 패킷 |
| 데이터 링크 계층 | 프레임 |
| 물리 계층 | 비트 |
