## SQL

### SELECT

SELECT는 테이블에서 원하는 컬럼을 조회할 때 사용한다.

```sql
SELECT *
FROM ANIMAL_INS;
```

### WHERE

WHERE는 조회할 데이터의 조건을 지정할 때 사용한다.

```sql
SELECT *
FROM ANIMAL_INS
WHERE ANIMAL_TYPE = 'Dog';
```

### ORDER BY

ORDER BY는 조회 결과를 특정 컬럼 기준으로 정렬할 때 사용한다.

```sql
SELECT *
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```

## CS

### 운영체제란?

운영체제는 컴퓨터의 하드웨어 자원을 관리하고, 사용자가 프로그램을 실행할 수 있도록 도와주는 시스템 소프트웨어다.

운영체제는 CPU, 메모리, 파일, 입출력 장치 등을 관리하며 여러 프로그램이 안정적으로 실행될 수 있게 한다.

## DB / Data Engineering

데이터베이스는 데이터를 체계적으로 저장하고 관리하는 시스템이다.

관계형 데이터베이스에서는 데이터를 테이블 형태로 저장하며, 테이블은 행과 컬럼으로 구성된다.

ETL은 데이터를 수집하고, 필요한 형태로 변환한 뒤, 저장소에 적재하는 과정이다.

- Extract: 데이터 수집
- Transform: 데이터 정제/변환
- Load: 데이터 저장
