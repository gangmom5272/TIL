## SQL

### SELECT
SELECT는 테이블에서 원하는 컬럼을 조회할 때 사용

### FROM
FROM은 어떤 테이블에서 데이터를 조회할지

### WHERE
WHERE는 조회할 데이터의 조건을 지정할 때 사용

### ROUND()
ROUND함수는 지정한 자리에서 반올림할 때 사용

### 집계함수
여러 행의 데이터를 입력받아 단일 값으로 요약하는 함수

### AS
출력되는 결과물의 컬럼명으로 별칭을 붙임

```sql
SELECT ROUND(AVG(DAILY_FEE), 0) AS 'AVERAGE_FEE'
FROM CAR_RENTAL_COMPANY_CAR
WHERE CAR_TYPE = 'SUV'
```


### ORDER BY
ORDER BY는 조회 결과를 특정 컬럼 기준으로 정렬할 때 사용, 먼저오는 조건으로 정렬하다 값이 같을 때 다음 조건을 사용함

### DESC
내림차순 정렬

### ASC
올림차순 정렬 생략 가능

```sql
SELECT FLAVOR
FROM FIRST_HALF
ORDER BY TOTAL_ORDER DESC, SHIPMENT_ID ASC
```

## DB / Data Engineering

데이터베이스는 데이터를 체계적으로 저장하고 관리하는 시스템이다.

관계형 데이터베이스에서는 데이터를 테이블 형태로 저장하며, 테이블은 행과 컬럼으로 구성된다.

ETL은 데이터를 수집하고, 필요한 형태로 변환한 뒤, 저장소에 적재하는 과정이다.

- Extract: 데이터 수집
- Transform: 데이터 정제/변환
- Load: 데이터 저장
